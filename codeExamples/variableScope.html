Scope determines the accessibility (visibility) of variables for use in the JavaScript Code.

In JavaScript there are two types of scope:

  • Local scope
  • Global scope
JavaScript has function scope which means that each function creates a new scope within that function.Generally you try to avoid using global variables whenever you can, but you will see quite qickly that some items need to be Global.
